On the Subject of Wind-Up Grommets

Rusty Dusty Four-hole Trouble

/!\ IMPORTANT /!\
This is a module in the Wind-Up series. Refer to Appendix WIND-UP KEY for important notes.

  • The module holds 4 key grommets. Three in a row above the fourth.
  • The top 3 pegs can be turned between 4 positions. The bottom peg can be turned to submit the current combo.
  • Each peg, as well as the washer around the Submit Peg, can have 1 of 3 materials, each with a value:
Copper Brass Steel
1 2 3

Follow these rules in the order they are listed. Perform the first action that applies. Any values below 1 or above 4 loop back around:

  1. All 3 Turn Pegs are the same material
    • If the Submit Peg is the same material as the Submit Washer
      • Right Turn Peg should be set to its own material, then subtract one going left
    • ELSE
      • Invert the values (copper = 4). Left Turn Peg should then be set to it's own value, then add one going right
  2. There's more copper parts than brass or steel
    • Set each copper Turn Peg to the Submit Washer plus the Submit Peg. All other Turn Pegs should be their material plus their position
  3. There's only 2 materials used
    • Copper - Number of batteries, mod 4, +1
    • Brass - 3rd + 6th serial number, mod 4, +1
    • Steel - Number of ports, mod 4, +1
  4. All 3 Turn Pegs are different materials
    • Copper - Copper + its position
    • Brass - left peg + right peg
    • Steel - Steel + Submit Washer
  5. If none of the above apply, Take the value of each Turn Peg and add the Submit Washer

Appendix WIND-UP KEY

Keep your friends close, and your keys closer...

This section goes over the mechanics of Wind-Up Key modules

  • Only one key will appear in the mission
  • It will always spawn on the first bomb
  • The key must be used across multiple modules, across multiple bombs, when applicable
  • If the key is already placed in a hole, you must pick it up before moving it to another hole

DO NOT LOSE YOUR KEY
YOU HAVE BEEN WARNED

Note: due to the nature of these modules, controllers are not recommended, but the support is there
